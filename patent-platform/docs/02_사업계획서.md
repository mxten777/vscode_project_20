## 1. 사업 개요

### 1.1 사업명
**PatentPlatform** — 한국 특허 모니터링 자동화 SaaS

### 1.2 사업 목표
KIPRIS 공공데이터를 활용해 **특허 수집 자동화, 맞춤 알림, 분석 리포트**를 제공하는 구독형 B2B SaaS를 구축·운영한다.

### 1.3 핵심 가치 제안

> *"방대한 특허 데이터를 실시간으로 모니터링하여, 기업 IP 전략 담당자가 경쟁에서 한 발 앞서 대응할 수 있도록 한다."*

| 기존 방식 | PatentPlatform |
|-----------|---------------|
| KIPRIS 웹사이트 수동 검색 | 매일 자동 수집·업데이트 |
| 키워드별 메일 수신 (노이즈 많음) | 정밀 규칙 기반 필터링 알림 |
| 엑셀 수작업 집계 | 실시간 차트·통계 리포트 |
| 개인별 분산 관리 | 조직 단위 협업, 역할 관리 |

## 2. 시장 분석

### 2.1 목표 시장

**TAM (Total Addressable Market)**
- 국내 IP 관리 소프트웨어 시장: 약 **1,200억 원** (2025년, 한국IP전략진흥원)
- 글로벌 IP 관리 소프트웨어 시장: 약 **15조 원** (2025년, Grand View Research)

**SAM (Serviceable Addressable Market)**
- 국내 중견·대기업 연구소 IP 담당, 특허법인, 스타트업 기술팀
- 추정 기업 수: 약 **8,000개 사** (연 특허 출원 5건 이상 기업)

**SOM (Serviceable Obtainable Market)**
- 초기 3년 목표: **400개 사** 확보 (SAM의 5%)
- 월 평균 ARR 기준 추정 매출: **월 2,400만 원 이상**

### 2.2 고객 세그먼트

| 세그먼트 | 특징 | 주요 니즈 |
|---------|------|-----------|
| **대기업 IP 팀** | 연간 수백 건 출원 관리 | 대규모 데이터 처리, API 연동 |
| **중소·벤처 연구소** | IP 전담 인력 부족 | 비용 효율적, 간편 사용성 |
| **특허법인·대리인** | 다수 고객사 포트폴리오 관리 | 멀티 조직 대시보드 |
| **스타트업** | 기술 차별화 검증 필요 | IPC 분석, 경쟁사 추적 |
| **대학·연구기관** | 기술이전 기회 탐색 | 만료 특허 탐지 |

### 2.3 시장 트렌드

- **생성형 AI 특허 급증**: 2023~2025년 AI 관련 특허 출원 전년 대비 연 35% 증가
- **글로벌 특허 분쟁 증가**: 특허 침해 소송 비용 대기업 평균 100억 원 이상
- **IP 규정 강화**: 기술보호법 개정으로 중소기업 특허 관리 의무 강화 추세
- **공공데이터 개방**: KIPRIS Open API 데이터 품질 및 범위 지속 확대

## 3. 제품 및 서비스

### 3.1 제품 구성

```
PatentPlatform v1.0
│
├── 특허 검색·조회 모듈
│   ├── 키워드·출원인·IPC 통합 검색
│   ├── 다중 필터 (상태/날짜/기술분류)
│   └── 특허 상세 (초록·IPC·상태·발명자)
│
├── 주시 목록 모듈
│   ├── 원클릭 등록/해제
│   ├── 주시 메모
│   └── D-Day 만료 카운트다운
│
├── 알림 규칙 모듈
│   ├── 경쟁사 / IPC / 키워드 / 만료 임박
│   ├── 이메일·카카오알림톡 채널
│   └── 규칙 ON/OFF 토글
│
├── 분석 리포트 모듈
│   ├── 기간별 출원 추이 라인 차트
│   ├── 상태별 분포 파이 차트
│   ├── 출원인 TOP10 바 차트
│   └── 만료 임박 리스트
│
└── 자동화 인프라
    ├── 매일 KIPRIS 신규 특허 수집
```

## 4. 비즈니스 모델

### 4.1 수익 모델

**구독 SaaS (MRR 기반)**
- Free → Pro → Enterprise 단계적 업셀링
- 연간 결제 시 2개월 무료 (17% 할인)

**추가 수익원**
- 🔹 **API 사용량 과금**: Enterprise 고객 대상 API 호출 건당 과금
- 🔹 **데이터 내보내기**: CSV/XLSX 대량 내보내기 유료화
- 🔹 **컨설팅 서비스**: 특허맵·경쟁분석 보고서 제작 (건당 수수료)

### 4.2 수익 전망

## 5. 경쟁 우위

### 5.1 경쟁사 분석

| 경쟁사 | 강점 | 약점 | 우리의 우위 |
|--------|------|------|------------|
| KIPRIS Plus (특허청 공식) | 데이터 정확성 | UI 노후, 알림 없음 | 자동화·알림·협업 |
| Winniz (윈니즈) | 국내 특허 분석 | 고가, 대기업 전용 | 중소기업 가격 접근성 |
| Darts-ip (글로벌) | 글로벌 특허 커버리지 | 한국어 미흡, 고가 | 한국어 최적화 |
| 자체 엑셀 관리 | 무료 | 수작업, 실시간성 없음 | 자동화 효율성 |

### 5.2 차별화 포인트

1. **KIPRIS 공공데이터 직접 파싱** — 매일 최신 데이터 자동 동기화
2. **한국 특허 UI/UX 최적화** — 한글 IPC 코드명, 한국 출원인명 처리
3. **조직 단위 협업** — 팀원 초대, 역할(admin/member) 관리
4. **Vercel 서버리스 인프라** — 초기 인프라 비용 최소화, 수요 급증 시 자동 스케일
5. **목업 모드** — Supabase 없이도 즉시 데모·영업 가능

## 6. 운영 계획

### 6.1 팀 구성 (목표)

| 역할 | 인원 | 주요 업무 |
|------|------|-----------|
| CTO/풀스택 개발 | 1 | 플랫폼 개발·운영 |
| 마케팅/세일즈 | 1 | B2B 영업, 콘텐츠 마케팅 |
| IP 도메인 전문가 | 1 (파트타임) | 특허 데이터 품질 검증, 고객 지원 |

### 6.2 고객 획득 전략

**Inbound (콘텐츠 마케팅)**
- 「특허 만료 기술 트렌드」 월간 무료 리포트 발행 (이메일 리드 확보)
- 기술 블로그: KIPRIS API 활용법, 특허 분석 팁

**Outbound (B2B 영업)**
- 중소벤처기업진흥공단, 창조경제혁신센터 연계 스타트업 대상 Free 체험 제공
- 특허법인·변리사 사무소 파트너십 (레퍼럴 수수료 10%)

**채널**
- Product Hunt 론칭
- 네이버 스마트스토어·브랜드 페이지
- 특허청·한국IP전략진흥원 공공 파트너십

### 8.2 손익분기점

- **BEP**: Pro 구독자 약 **2명** (월 9만 원 ÷ Pro 4.9만 원)으로 인프라 비용 회수
- **인건비 BEP**: 개발자 1인 월급 500만 원 기준 Pro **104명**, Enterprise **26건** 조합 시 달성

| 단계 | 금액 | 용도 |
|------|------|------|
| Pre-seed | 1억 원 | 팀 확충(마케터), 유료 광고, 법인 설립 |
| Seed | 5억 원 | 기술 고도화(AI), 영업 채널 확대 |
